# Fontes — Hanken Grotesk (design system)

Coloque aqui os arquivos `.ttf` (carregados em runtime pelo `main.rs`):

- HankenGrotesk-Regular.ttf    (400)
- HankenGrotesk-Medium.ttf     (500)
- HankenGrotesk-SemiBold.ttf   (600)
- HankenGrotesk-Bold.ttf       (700)
- HankenGrotesk-ExtraBold.ttf  (800)

Opcional: HankenGrotesk-wght-.ttf (variável). Os pesos estáticos acima
cobrem os usos do sistema (500/700/800).
